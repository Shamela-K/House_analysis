Project Overview

This project analyzes luxury housing market trends using a complete data analytics pipeline. The goal is to understand booking patterns, builder performance, buyer preferences, and geographical distribution of luxury housing projects.

The project integrates Python for data preprocessing, MySQL for database storage, and Power BI for data visualization, enabling interactive dashboards and meaningful business insights.

Project Objectives
Analyze booking trends across different micro-markets and quarters
Identify top-performing builders based on revenue and ticket size
Evaluate the impact of amenities on booking conversion rates
Understand buyer preferences for housing configurations
Analyze the efficiency of different sales channels
Provide geographical insights into luxury housing project distribution
Data Pipeline

Raw Dataset (CSV File)
→ Data Loading using Python (Pandas)
→ Data Cleaning (Handling Null Values, Removing Unnecessary Columns)
→ Feature Engineering (Creating Amenity Score, Booking Conversion Rate, Quarter Column)
→ Data Encoding (Label Encoding & One-Hot Encoding)
→ Data Validation and Preparation
→ Loading Processed Data into MySQL Database
→ SQL Queries for Aggregation and Analysis
→ Connecting Power BI to MySQL Database
→ Creating Interactive Dashboards and Visualizations
→ Generating Business Insights and Final Report

Key Business Insights
Certain micro-markets show consistently higher booking demand.
Some builders dominate the luxury housing market in terms of revenue.
Projects with better amenities tend to attract more buyer interest.
Larger configurations such as 3BHK and 4BHK are highly preferred by buyers.
Sales channel performance varies significantly in converting leads into bookings.
Luxury housing projects are concentrated in specific high-growth areas.
